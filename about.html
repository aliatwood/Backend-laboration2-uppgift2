Om webbplatsen

Denna webbplats visar arbetserfarenheter hämtade från ett REST API byggt med Node.js, Express och MariaDB.

Du kan se alla erfarenheter på startsidan, lägga till nya via formuläret och ta bort befintliga poster.

Databas

Databasservern som används är MySQL/MariaDB, hostad på Railway.

Slutsatser

Uppgiften gav en god förståelse för hur REST API:er fungerar och hur man kan kommunicera mellan en frontend och backend via Fetch API. Det var lärorikt att se hur data flödar från databasen till användargränssnittet.
